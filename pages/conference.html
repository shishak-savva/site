WeeBee Конференция

Данное приложение позволяет проводить видеоконференции (камера, микрофон, показ экрана) в хорошем качестве бесплатно. Имеет самый обычный чат и позволяет включить интерактивную доску с презентацией (кнопка "демонстрация"). Сама видео свзязь работает через сервер Media WeeBee.

Помимо звонков в браузере, может принимать звонки по протокору SIP, т.е. можно попасть в комнату и общаться с её участниками просто ПОЗВОНИВ ПО ТЕЛЕФОНУ горячей линии "Альфаматика" с просьбой перевести звонок на номер 234

Технологии сервера:

  • Koa.js
  • Axios (Обращение к Media WeeBee)
Технологии сервера телефонии:
  • SIP/UDP
  • Socket.io client
  • Axios
Технологии клиента:
  • Typescript
  • Axios
  • React
  • Pullstate
  • React-konva
