Сервер синхронизации рекламы, бегущих строк и новостей

Это сервер, который позволяет синхронизировать вопросизведение рекламы на нескольких устройствах. Он по очереди рассылает на устройства сигналы для смены рекламного ролика и позволяет менять очередь прямо на ходу. Данный сервер кэширует файлы для рекламы и рассылает их на устройства.

Технологии (сервер):

  • Go lang
  • GRPC
  • Gorilla Websocket
Клиентская библиотека:
  • Typescript
  • Npm (сделал из этого npm пакет)
  • React
  • Websocket
