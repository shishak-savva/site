Оператор электронной очереди для Аэрофлота

Приложение является частью системы новой системы управления электронной очередью. Через данное приложение операторы принимают клиентов электронной очереди. Здесь же они получают инструкции по обслуживанию для каждого клиента. Приложение позволяет операторам просматривать свою статистику и статистику всего офиса. Так же они могут вставать на паузу. Во время обслуживания клиента они через данное приложение могут вернуть его в очередь, перенаправить на другое рабочее место или на конкретного оператора, а так же добавить услуги для обслуживания.

Технологии:

  • TypeScript
  • React
  • Vite
  • Pullstate
  • Ant UI
  • React Query
  • Sass
  • Axios
  • Electron
