Media WeeBee сервер

С помощью данного сервера, можно встроить видеосвязь на любой сайт. Данный сервер расчитан на большое количество пользователей. Он использует библиотеку Mediasoup, поэтому работает по архитектуре SFU, что обеспечивает бесперебойную работу при большом количестве подключений на сервер.

Если перейти к конкретике, с таким сервером, 1 ядро машины может спокойно обрабатывать 400-500 точек получения видеосвязи. В данном сервере уже реализована многоядерная архитектура передачи трансляций. Мало того, на данном сервере реализована многохостинговая архитектура. Что позволяет ядрам с разных машин обмениваться трансляциями. Поэтому одну комнату могут обслуживать сразу несколько машин и 1 машина может обслуживать несколько комнат.

Сама связь проходит по двум протоколам: RTP и WebRTC. Помимо трансляций из браузера, сервис позволяет подключать к конференции трансляции из других источников, например: файловая система, SIP телефония, TV телевидение и прочие источники поддерживающие RTP протокол.

У него есть встроенная система защиты на уровне API, позволяющая в пределах комнаты каждому пользователю получать только те видео треки, на которые у него есть разрешения. А также, под контролем управляющего сервиса, позволяет входить в комнату в режиме невидимки.

  • Koa.js
